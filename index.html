Apa Itu HTML?

HTML adalah sebuah bahasa markup yang digunakan untuk membuat sebuah halaman web dan menampilkan berbagai informasi di dalam sebuah browser.
HTML berisi kode-kode tag yang menginstruksikan browser untuk menghasilkan tampilan sesuai dengan yang diinginkan. HTML saat ini merupakan
standar Internet yang didefinisikan dan dikendalikan penggunaannya oleh World Wide Web Consortium (W3C).

A. Bagaimana Sejarah HTML?

Berikut perkembangan HTML dari tahun ke tahun:
  • HTML 2.0, 22 September 1995
  • HTML 3.2, 14 Januari 1996
  • HTML 4.0, 18 Desember 1997
  • ISO/IEC 15445:2000 "ISO HTML", 15 Mei 2000
  • XHTML 1.0 -> gabungan HTML & XML, 24 Januari 2000
  • XHTML 1.1, 16 Januari 2007

    • B. Apa saja ciri-ciri dokumen HTML?

  • Ekstensi file berupa .htm atau .html
  • Non case sensitive
  • Terdiri atas tag-tag pembuka dan penutup (beberapa tak memiliki penutup)
  • Tag-tag saling berpasangan & bersarang

    • Tag HTML

    Tag HTML adalah elemen dasar dalam bahasa HTML (HyperText Markup Language) yang digunakan untuk membuat dan memformat konten di halaman web.
    Setiap elemen HTML didefinisikan oleh tag yang memiliki tujuan tertentu dalam struktur dan penampilan dokumen web.

    A. Penulisan Tag

    Penulisan tag terbagi menjadi 3 yaitu:
  • Tag Pembuka ( < > )
  • Tag Penutup ( ). Contoh : < p > ini paragraf < /p >
  • Single Tag ( < />. Contoh: < br />, < hr />.
